Welcome

Leeds.dart is a user group that meets in Leeds on the first Wednesday of every other month. We aim to create a community of Dart developers in Yorkshire and provide knowledgeable and informative talks about the Dart language and various Dart projects.

What makes Dart so great?

Dart offers a super tight development stack with great libraries and tools. Dart has a pure native version of the incredible Angular2 framework. With Flutter you can develop super slick cross-platform mobile apps. You can write server-side and client-side apps in Dart. The Dart language is familiar and predictable. Come join us and find out more reasons for yourself! If you're looking for more information about the Dart language then head over to the official dartlang website.
